# Arduino + Python Radar Emulator
Arduino radar emulator using ultrasonic sensor and servo motor. Python uses the serial port to read data from Arduino board and plots it in real-time to emulate a radar by approximating the distance of objects along the 180Â° axis of the servo motor

Full tutorial at: https://makersportal.com/blog/2020/3/26/arduino-raspberry-pi-radar
